# 都市インディーズ

## プロジェクトについて

東京の地下バンドをテーマにした、リズムゲーム「都市インディーズ」を開発しています。  
きっかけは、もともと絵を描くことも好きなので、「コードを書くこと」と「絵を描くこと」を組み合わせて、何か形になるものを作ってみたいと思いました。  
このプロジェクトを通して、
技術的な挑戦をするだけでなく、将来ライブハウスなどで音楽をやっている人たちと交流するきっかけにもなればいいなと思っています。

#### 開発状況
現在はまず基本的な仕組みを作ることを目標として、シンプルなデモを制作しています。  
将来的には、メニュー画面、東京のライブハウスマップ、バンド選択画面、楽曲選択画面などを追加する予定です。

ユーザーはメニューから東京のライブハウスマップに入り、興味のあるライブハウスを選択します。  
その後、そこで活動している地下バンドと楽曲を選び、ゲームを開始できる仕組みを想定しています。

## Demo
知り合いの地下バンドから「Cloudy」という曲の音源を提供してもらい、曲の世界観をもとにデモを制作しました。

#### 遊び方
バンドメンバーをモチーフにしたキャラクターを操作し、音楽のリズムに合わせて敵を撃つゲームです。

#### 技術的な背景
GitHub 上で公開されている既存の譜面エディタを見つけ、まずはそれを利用してプロトタイプを進めることにしました。  
現在使用しているエディタは BPM 固定を前提としていますが、今後はコード構造を理解した上で、BPM 変化に対応できるように改良することを考えています。


#### 今後の構想

現在は 1 曲のみのデモですが、将来的には複数の楽曲を追加していきたいと考えています。  
その場合、音声データや画像データの管理が必要になるため、ストレージの設計についても考えています。
