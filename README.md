# 都市インディーズ

## プロジェクトについて

東京の地下バンドをテーマにした、個人制作のリズムゲーム「都市インディーズ」を開発しています。
現在はまずゲームの基本的な仕組みを作ることを目標として、シンプルなデモを制作しています。

## Demo
知り合いの地下バンドから「Cloudy」という曲の音源を提供してもらい、
曲の世界観をもとにデモを制作しました。


#### 遊び方
バンドメンバーをモチーフにしたキャラクターを操作し、音楽のリズムに合わせて敵を撃つゲームです。

#### 技術的な背景

当初は、BPM が変化する楽曲に対応した譜面エディタを自分で一から実装することを考えていました。しかし、エディタ開発は想像以上に難しく、プロジェクトがしばらく停滞してしまいました。その後、GitHub 上で公開されている既存の譜面エディタを見つけ、まずはそれを利用してプロトタイプを進めることにしました。現在使用しているエディタは BPM 固定を前提としていますが、今後はコード構造を理解した上で、BPM 変化に対応できるように改良することを考えています。


#### 今後の構想

現在は 1 曲のみのデモですが、将来的には複数の楽曲を追加していきたいと考えています。
その場合、音声データや画像データの管理が必要になるため、ストレージの設計についても考えています。

例えば、AWS S3 を利用したassets管理の事例を調査していますが、個人プロジェクトとしてはコスト面も考慮する必要があるため、今後は他の方法も含めて検討していく予定です。
